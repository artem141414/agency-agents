# NEXUS — AI-агентство по подписке

> **Network of EXperts, Unified in Strategy.** Пользователь общается с командой ИИ-специалистов по отделам: скачивает лёгкое десктоп-приложение, входит онлайн (проверка подписки), подключает ключ любой LLM или работает на нашей модели — и получает готовые результаты.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Форк и развитие [msitarzewski/agency-agents](https://github.com/msitarzewski/agency-agents) — здесь библиотека агентов превращена в полноценную RU-платформу-агентство, адаптированную под рынок РФ-2026.

---

## Что это

NEXUS — это связка из трёх частей:

| Часть | Где | Что делает |
|---|---|---|
| **Клиент** | `marketing-office/` *(отдельная папка)* | Десктоп-приложение (Flask). Юзер качает, ставит, работает. 22 МБ, без Python, от 4 ГБ RAM |
| **Auth-сервер** | `nexus-auth-server/` *(отдельная папка)* | Единственный online-компонент: проверка подписки + прокси «Нашей модели». Не видит ключи и диалоги юзера |
| **Этот репозиторий** | `agency_agents/` | Источники агентов, стратегия/доктрина, планы развития, заметки |

> Код продукта (`marketing-office`, `nexus-auth-server`) лежит в соседних папках и версионируется отдельно. Здесь — **стратегия, планирование и библиотека агентов**.

## Что реализовано (платформа)

- **191 агент** по 12 отделам: Маркетинг, Инженерия, Дизайн, Продажи, Продукт, Аналитика, QA, Спецотделы, Игры/XR, Наука, **OSINT**, **Кибербезопасность**. Все на русском, под РФ-2026, с методологией обработки запроса.
- **Оркестратор** — задача в одну строку → план с проактивным выбором модели по сложности → команда агентов → синтез. Контроль токенов (лимит 8 агентов).
- **Мульти-LLM провайдер-слой** — своя модель (serverless DeepInfra/Qwen3) ИЛИ ключ любой LLM (Anthropic, OpenAI, DeepSeek, OpenRouter). Авто-эскалация дешёвая→сильная модель + оценка качества ответов.
- **Долговременная память** — профиль пользователя и контекст копятся между сессиями (по образцу hermes-agent: markdown + инъекция в промпт + tools).
- **AI Readiness Score** — оценка готовности задачи/брифа к работе команды (Pro).
- **Обработка файлов** — документы (PDF/Word/Excel/PowerPoint/текст → извлечение текста) и изображения (vision) в чате и оркестраторе.
- **Скачивание результатов** — агент отдаёт файлы (HTML-прототип, тексты) → zip с инструкцией.
- **Подписка Free/Pro** — гейтинг отделов и оркестратора. **Профиль**, **админка** (RBAC, ручная активация Pro, аудит).
- **Защита** — усиленный GUARD против prompt-injection; Guard от раскрытия внутренней кухни.

Подробный статус и роадмап — в [`launch/`](launch/) (планы 00–11) и заметках [`obsidian-notes/`](obsidian-notes/).

## Структура репозитория

```
agents/             — 191 агент (.md, плоский frontmatter, RU)
departments.json    — реестр отделов и подотделов
strategy/           — доктрина NEXUS, playbooks, runbooks, prompt-library
launch/             — планы развития 00–11 (стратегия → монетизация → фичи → backlog)
obsidian-notes/     — рабочие заметки и статус проекта
build/              — сборка агентов в плагины Claude Code (build-plugins.mjs)
integrations/       — генерация под разные AI-инструменты
```
