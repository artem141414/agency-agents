# NEXUS — Статус проекта

> Сводка по состоянию на **14 июня 2026**. Платформа-агентство + монетизация + расширения.

## Что такое NEXUS
AI-агентство: пользователь общается с командой ИИ-специалистов по отделам. Качает приложение, логинится онлайн (проверка подписки), **выбирает модель — нашу (serverless) или свой ключ любой LLM**, работает.

### Репозитории (на GitHub с 14 июня)
- **`github.com/artem141414/nexus`** — МОНОРЕПО продукта, рабочая папка `Desktop/projects/nexus/`:
  - `client/` — десктоп-приложение (Flask), то что качает юзер. 191 агент, оркестратор, фичи.
  - `auth-server/` — онлайн-сервер: авторизация подписки + прокси «Нашей модели» + админка.
- **`github.com/artem141414/agency-agents`** — форк msitarzewski/agency-agents: библиотека-источник агентов + стратегия + планы launch 00-11 + эти obsidian-заметки.

> Миграция выполнена: код продукта переехал из `marketing-office/` + `nexus-auth-server/` в единый `nexus/`. Старые папки — резервная копия, можно удалить. `~/.nexus/` (сессия/конфиг/память/проекты) — в домашней папке, не переезжает.

## ✅ Сделано

### Платформа и агенты
- **191 агент** по **12 отделам** (+OSINT, +Кибербезопасность к прежним 10). Все на русском, под РФ-2026, с методологией обработки запроса. Кибербез включает PentAGI-методологию; OSINT — легальная разведка по открытым источникам (граница «открытый Яндекс можно, закрытый пробив нельзя»).
- **Оркестратор** — задача → план с **проактивным выбором модели по сложности** → команда → синтез. Лимит 8 агентов.
- **Гибрид** — общий чат оркестратора ИЛИ прямой заход к агенту.

### Модели (мульти-LLM)
- **Провайдер-слой**: своя модель (serverless **DeepInfra Qwen3-32B**, работает — баланс пополнен) ИЛИ ключ любой LLM (Anthropic/OpenAI/DeepSeek/**OpenRouter**). `<think>`-теги reasoning-моделей вырезаются.
- **Авто-эскалация** дешёвая→сильная по качеству ответа + опц. AI-судья. **Оценка качества** (evaluate_answer) + статистика моделей.
- Российские провайдеры GigaChat/YaGPT — спроектированы (в плане).

### Фичи продукта
- **Долговременная память** (hermes A+B): профиль юзера + контекст копятся между сессиями (USER.md/MEMORY.md, инъекция в промпт, tool `remember`, авто-профиль). Видно/чистится в профиле.
- **AI Readiness Score** (Pro) — оценка готовности задачи/брифа к работе команды по 5 категориям + подсказки.
- **Обработка файлов** — документы (PDF/Word/Excel/PowerPoint/текст → извлечение) и изображения (vision) в чате И оркестраторе.
- **Скачивание результатов** — агент отдаёт файлы → zip с инструкцией.
- **Профиль** (выбор модели, расход, память), **админка** (RBAC, ручная активация Pro, аудит, `:8000/admin`), **панель «Последние чаты»** (ховер на лого, localStorage, FIFO 5).
- **Усиленный GUARD** против prompt-injection (из system_prompts топ-продуктов) + QUALITY_HINT.

### Монетизация / инфра
- Подписка Free/Pro (гейтинг по `FREE_DEPARTMENTS`). «Наша модель» = Pro.
- Платёжный checkout универсальный (любая ссылка в `NEXUS_CHECKOUT_URL`).

## Технические заметки
- Dev-доступы: auth `:8000`, клиент `:5000`, демо `demo@nexus.ru`/`demopass123`, суперадмин `admin@nexus.ru`/`adminpass123`.
- Запуск из репозитория:
  - auth: `cd nexus/auth-server && ./venv/Scripts/python.exe server.py`
  - клиент: `cd nexus/client && NEXUS_AUTH_URL=http://127.0.0.1:8000 NEXUS_LLM_URL=http://127.0.0.1:8000/api/llm ./.venv/Scripts/python.exe app.py`
- DeepInfra-ключ в `nexus/auth-server/.env` (в gitignore, не в репо). Проверка ключей: `python verify_keys.py`.
- Эталон стиля агента: `client/agents/vk-strategist.md`. Плоский frontmatter (парсер построчный). Реестр: `client/departments.json`.
- Модули клиента: `app.py` (роуты/оркестратор/память/readiness/вложения), `llm_providers.py` (мульти-LLM+эскалация), `file_extract.py` (документы). Сервер: `server.py` (auth+прокси), `admin.py` (админка).
- Секреты НЕ в git: `.env`, `*.db`, venv, сборки — в `.gitignore` обоих репо.

См. операционную память проекта (`memory/nexus-platform-evolution.md` и др.).
